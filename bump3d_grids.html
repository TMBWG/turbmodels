Grids - 3D Bump-in-channel Verification Case

A series of 5 nested 3-D grids are provided. Each coarser grid is exactly every-other-point of the next finer grid, ranging from the finest 65 x 705 x 321 to the coarsest 5 x 45 x 21 grid. The finest grid has minimum spacing at the wall of y=1 x 10-6, giving an approximate average y+=0.12 over the plate at the Reynolds number run. Even the coarsest grid has reasonably fine wall-normal spacing, giving an approximate average y+=2.0 over the plate. The grid is stretched in the wall-normal direction, and is also clustered near the plate leading and trailing edges. The spacing in the spanwise direction is uniform. STRUCTURED VERSIONS OF GRIDS

PLOT3D Files

Due to file size limitations, the PLOT3D files are given as 2-D (formatted, MG, 2D, nbl=1) files, and a specifically-hardwired Fortran90 program is provided to translate the 2-D grid to the 3-D version.

Download the 2-D version of the grids in PLOT3D format here:

  • (2-D) 705 x 321 (gzipped) (2 MB)
  • (2-D) 353 x 161 (gzipped) (< 1 MB)
  • (2-D) 177 x 81 (< 1 MB)
  • (2-D) 89 x 41 (< 1 MB)
  • (2-D) 45 x 21 (< 1 MB)

If desired, example Neutral Map Files associated with 3-D version of the grids after conversion are given here (these files specify grid indices associated with each boundary condition - see The Neutral Map File):

  • Neutral Map File for (3-D) 65 x 705 x 321
  • Neutral Map File for (3-D) 33 x 353 x 161
  • Neutral Map File for (3-D) 17 x 177 x 81
  • Neutral Map File for (3-D) 9 x 89 x 41
  • Neutral Map File for (3-D) 5 x 45 x 21

 

UNSTRUCTURED VERSIONS OF GRIDS

CGNS and AFLR3 Files

The unstructured versions of the same grids are all given as CGNS and AFLR3 (UGRID) files. However, due to file size limitations, the finest 65x705x321 grid is not available here. Note that the BCs written inside the CGNS files are for guidance only, and may not reflect the appropriate BCs needed for your application. To read/write CGNS files, (free) software may be necessary if your pre/post-processing tools do not already handle it. See: CGNS Website for details.

The unstructured grids are all given two different ways: as hexahedra, or cut into tetrahedra.

Download the unstructured CGNS grids (gzipped) as 3-D hexahedra here:

  • Unstructured quad version of 33 x 353 x 161 3D-bump grid (35.8 MB)
  • Unstructured quad version of 17 x 177 x 81 3D-bump grid (4.3 MB)
  • Unstructured quad version of 9 x 89 x 41 3D-bump grid (< 1 MB)

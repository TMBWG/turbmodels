2D Zero Pressure Gradient Flat Plate Validation Case

K-e-Rt Model Results
 

Link to K-e-Rt equations

Above K-e-Rt results are from CFD++ on the 545x385 grid. CFD++ used freestream turbulence intensity=1.0% and freestream turbulent viscosity (relative to laminar)=20. Please read note 5 on Notes on running CFD page. The right-hand plot is extracted from the location where Retheta=10000. Note that these are compressible code results at "essentially incompressible" conditions of M=0.2. There may be a very small influence of compressibility. The CFD postprocessed data files are given here for reference: kert-cf_cfd++.dat and kert-upyp_cfd++.dat.

Note that this model is currently assigned MRR Level 1 for the purposes of this website. This is because at this time the TMBWG has results for this model from only one code. The results on this page are therefore not necessarily reliable until additional independent code(s) can be used to verify the model implementation.

DNS: Supersonic Isothermal-Wall Channel Flow

Return to: Data from DNS - Intro Page

Return to: Turbulence Modeling Resource Home Page


 

The data on this page were provided by G. Coleman.

This DNS studied compressible supersonic turbulent flow in a plane channel with isothermal walls. Mach numbers (based on bulk velocity and sound speed at the walls) were 1.5 and 3.0. Reynolds numbers (based on centerline velocity and channel half-width) were approximately 3000. Additional details concerning the data, code, and computational methodology can be found in:

  • Coleman, G. N., Kim, J., and Moser, R. D., "A numerical study of turbulent supersonic isothermal-wall channel flow," J. Fluid Mech, Vol. 305, 1995, pp. 159-183, https://doi.org/10.1017/S0022112095004587.

The data files include mean and turbulence statistics. The DNS data files are provided below:

  • Data for M=1.5 case
    • Auxiliary/Reynolds-stress budget data for M=1.5 case (some data repeated)
  • Data for M=3.0 case
    • Auxiliary/Reynolds-stress budget data for M=3 case (some data repeated)

This represents selected data files from the U of Illinois compressible shear layer database.
Some items were left out (including instantaneous velocity data and Schlieren images and movies), 
in order to make the file size manageable.

As of 03/28/2025, the full set of data can be found at:

https://uofi.box.com/s/vibg76guf44oxkizz310piq76c7dvt5b

TURBULENCE MODEL NUMERICAL ANALYSIS

Grids - 3D ONERA M6 Wing

A paper describing the grid generation for this case is AIAA-2018-1101, https://doi.org/10.2514/6.2018-1101. For the ONERA M6 wing, a set of Fortran programs are provided to create both unstructured and structured grids. These grids are normalized so that the root chord is unity.

    wing_release_072319.tar.gz <- updated on 07/29/2019 (More spacing controls, blunt tip wing, generate 2D grid files. Minor bugs fixed for the coarsening code.)

The gzipped tar file above includes both a grid generation program as well as a coarsening program, with sample inputs and readme file.
 

Recommended input files for large-size grids are:

  • input.nml_wing_tetra_stt for Tetrahedral grids
  • input.nml_wing_strct_stt for Structured grids
(Note: input files need to be renamed as input.nml when used. Set generate_su2grid_file = T to write .su2 file.)
 

Recommended input files for coarsening the grids (to be used with the coarsening program) are:

  • input_coarsen.nml_wing_tetra_stt for Tetrahedral grids <- updated on 04/30/2018
  • input_coarsen.nml_wing_strct_stt for Structured grids <- updated on 04/30/2018
(Note: input files need to be renamed as input_coarsen.nml when used. Set generate_su2grid_file = T to write .su2 file.)
 
      (A second set of input files is also provided for increased number of surface points along with significantly reduced LE/TE stretching):
      • input.nml_wing_tetra_2_stt for Tetrahedral grids
      • input.nml_wing_strct_2_stt for Structured grids

      (The same coarsen input files apply.)
